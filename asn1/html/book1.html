Basics of Web Design

Webpage. Technology.

Overview

Basics of Web Design: HTML5 & CSS (Terry Felke-Morris) is a comprehensive, beginner-friendly textbook providing a balanced foundation of "hard" skills (HTML5, CSS3) and "soft" skills (design principles, accessibility, SEO) for web development. It focuses on modern techniques like responsive design, Flexbox, and Grid, featuring hands-on exercises, case studies, and practical, two-page chapter layouts.
